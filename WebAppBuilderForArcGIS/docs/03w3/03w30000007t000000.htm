Layer List widget

The Layer List widget provides a list of 2D or 3D layers in the scene.

Configuring the Layer List widget

The Layer List widget can be set to open automatically when apps start. To do so, click the dot on the widget to turn it dark green. There is no configuration for the Layer List widget.

Using the Layer List widget

Steps:
  1. Click the widget to open it.
  2. Click the check boxes to turn layers on and off in the scene.
  3. Optionally, hover the layer and click the Go to full extent button on the right to zoom your scene to the layer's full extent.
