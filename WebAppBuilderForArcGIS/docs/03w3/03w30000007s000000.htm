Navigate widget

Use the Navigate widget to rotate and pan the scene.

Configuring the Navigate widget

There is no configuration for the Navigate widget.

Using the Navigate widget

Steps:
  1. Click the Pan button and hold down the mouse button while you drag the scene in the direction you want to pan.
  2. Alternatively, click the right mouse button to pan the scene if you have a two-button mouse.
  3. Click the Rotate button again to return to the default behavior.
