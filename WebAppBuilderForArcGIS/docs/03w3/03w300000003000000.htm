Open a widget at app start

Set the openAtStart attribute to true when you want to open a widget when the application starts. The default value is false. The attribute only applies to in-panel widgets. 

"openAtStart": true,

If more than one widget in widgetOnScreen is set to true, the first one opens.

If more than one widget in the widget pool is set to true, the widget controller defines how they open.

See the config-readme.text file under the stemapp folder for details.
