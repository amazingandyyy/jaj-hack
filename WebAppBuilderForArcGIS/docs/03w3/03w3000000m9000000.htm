Near Me

The Near Me widget allows you to find features within a buffer of a defined address or location, view more detailed information about those features, and get directions to the selected feature, if desired.

Additionally, the Near Me widget can be configured to find polygon features containing a defined address or location, instead of polygon features within the buffer.

The following table shows the descriptions of the configurable attributes:

Attribute

Description

fontColor

String. Default value is #000000. This is the color displayed in the title section of the search results.

bufferInfo

Object. The properties of the array item are as follows:

  • isVisible—Boolean
  • Default value is false
  • Use the value true to display the buffer area on the map.

zoomToFeature

Boolean. Default value is false. Use the value true to zoom to the selected feature rather than the buffer extent.

intersectSearchedLocation

Boolean. Default value is false. Use the value true to return the polygon or polygons containing the searched location rather than polygons within the buffer.

defaultBufferDistance

Number. Default value is 5. Define the default value of the buffer slider.

bufferDistanceUnit

Object. The properties of the array item are as follows:

  • bufferUnit—String. Default value is UNIT_STATUTE_MILE. Unit of measurement used for the buffer size.
  • routeUnit—String. Default value is MILES. Unit of measurement used for routing.
  • distanceUnit—String. Default value is MILES. Unit of measurement used when reporting distances.
  • label—String. Default value is MILES. Used as a long label for distances.
  • acronym—String. Default values is MILES. Used as a short label for distances.
  • value—String. Default value is MILES.

directionLengthUnit

Object. The properties of the array item are as follows:

  • bufferUnit—String. Default value is UNIT_STATUTE_MILE. Unit of measurement used for the buffer size.
  • routeUnit—String. Default value is MILES. Unit of measurement used for routing.
  • distanceUnit—String. Default value is miles. Unit of measurement used when reporting distances.
  • label—String. Default value is MILES. Used as a long label for distances.
  • acronym—String. Default values is mi. Used as a short label for distances.
  • value—String. Default value is miles.

symbols

Object. The properties of the array item are as follows:

  • bufferSymbol
    • Object. The properties of the array item are as follows:
      • Color—List of R, G, B, a values for the symbol fill. Default value is [255,0,0,0]
      • Outline—Object. The properties of the array item are as follows:
        • Color—List of R, G, B, a values for the symbol outline. Default value is [255,189,1,255]
        • Width—Number. Default value 2.25. Width of the outline.
        • Type—String. Default value is esriSLS. The name of the symbol type to use for the outline.
        • Style—String. Default value is esriSLSSolid. The name of the style library to which the outline symbol belongs.
      • Type—String. Default value is esriSFS. The name of the symbol type to use for the fill.
      • Style—String. Default value is esriSFSSolid. The name of the style library to which the fill symbol belongs.
  • routeSymbol
    • Object. The properties of the array item are as follows:
      • Color—List of R, G, B, a values for the route symbol. Default value is [21,99,184,255]
      • Width—Number. Default value 3.75. Width of the route symbol.
      • Type—String. Default value is esriSLS. The name of the symbol type to use for the line.
      • Style—String. Default value is esriSLSSolid. The name of the style library to which the line symbol belongs.
  • graphicLocationSymbol
    • Object. The properties of the array item are as follows:
      • Angle—Number. Default value is 0. The angle of the symbol used to mark the searched location.
      • X-Offset—Number. Default value is 0. The x offset of the symbol used to mark the searched location.
      • Y-Offset—Number. Default value is 12. The y offset of the symbol used to mark the searched location
      • Type—String. Default value is esriPMS. The type of symbol used to mark the searched location.
      • URL—String. Default value is http://static.arcgis.com/images/Symbols/Basic/RedStickpin.png. URL to the graphic to use to mark the searched location.
      • imageData—String. 64 bit encode hash string of the symbol used to mark the searched location.
      • contentType—String. Default value is image/png. Type of image that may be configured for the searched location symbol.
      • Width—Number. Default value is 22.5. Width of the searched location symbol.
      • Height—Number. Default value is 22.5. Height of the searched location symbol.
  • polygonSymbol
  • Object. The properties of the array item are as follows:
    • Color—List of R, G, B, a values for the symbol fill. Default value is [255,0,0,0]
    • Outline—Object. The properties of the array item are as follows:
      • Color—List of R, G, B, a values for the symbol outline. Default value is [255,189,1,255]
      • Width—Number. Default value 2.25. Width of the outline.
      • Type—String. Default value is esriSLS. The name of the symbol type to use for the outline.
      • Style—String. Default value is esriSLSSolid. The name of the style library to which the outline symbol belongs.
    • Type—String. Default value is esriSFS. The name of the symbol type to use for the fill.
    • Style—String. Default value is esriSFSSolid. The name of the style library to which the fill symbol belongs.
  • polylineSymbol
    • Object. The properties of the array item are as follows:
      • Color—List of R, G, B, a values for the polyline symbol. Default value is [21,99,184,255]
      • Width—Number. Default value 3.75. Width of the polyline symbol.
      • Type—String. Default value is esriSLS. The name of the symbol type to use for the outline.
      • Style—String. Default value is esriSLSSolid. The name of the style library to which the outline symbol belongs.

showLocationTool

Boolean. Default value is false. Use the value true to add a button to the search bar that must be clicked to set a search location by clicking the map.

isGeodesic

Boolean. Default value is false. Use the value true to draw buffer and measure distances using geodesic distances.

searchSourceSettings

Object. The properties of the array item are as follows:

  • allPlaceHolder—String. Default value is "". Default placeholder text that displays in the search box.
  • showInfoWindowOnSelect—Boolean. Default value is true. Show pop-up for the found feature or location.
  • sources—List of layers to include in the results of the search for nearby features.
