Basemap Gallery widget

The BasemapGallery widget displays a collection images representing basemaps from your organization or a user-defined set of map or image services. When a new basemap is selected from the BasemapGallery, the map's basemap layers are removed and replaced with the basemap layers of the associated basemap selected in the gallery.

Configure the Basemap Gallery widget

There is no configuration for the Basemap Gallery widget. It synchronizes with the Basemap Gallery set in your organization.

Use the Basemap Gallery widget

Steps:
  1. Clicking the Basemap Gallery widget displays all the basemaps. Clicking one of the basemap thumbnails sets it as the active basemap for the app.
