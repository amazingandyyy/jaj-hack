Deploy your widget

To deploy your widget, follow these steps:

  1. Open manifest.json file in your custom widget folder, which describes widget properties. See Widget manifest.
  2. Modify the properties accordingly.
  3. Save the file.
  4. Copy your widget folder and paste it in the widgets repository. By default, the widgets repository is located under \client\stemapp\widgets folder for 2D app and \client\stemapp3d\widgets folder for 3D app. In this way, your widget is available to the builder. If you prefer to deploy the widget to a specific app only, you can copy the widget folder and paste it in the stemapp\widgets or stemapp3d\widgets folder within a downloaded app, then configure it in the app config file.
