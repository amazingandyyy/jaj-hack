Coordinate

The Coordinate widget displays x, y-coordinates dynamically when the pointer moves around the map. By default, the coordinate values are based on the spatial reference of the first basemap.

The following table shows the descriptions of the configurable attributes.

Attribute

Description

spatialReferences

Object[]; There is no default. Shows a pop-up menu containing all configured spatial references when the mouse hovers over the widget.

  • wkid—Number; There is no default. Well-known ID. Refer to Coordinate system IDs for details..
  • label: String. There is no default. A label for the specified WKID.
  • transformationWkid: Number. There is no default. The well-known ID {wkid:number} for the datum transformation to apply on projected geometries.
  • transformationLabel: String. There is no default. The well-known text {wkt:string} for the datum transformation to apply on projected geometries.
  • options: Object; There is no default; The auxiliary information of Spatial Reference.
    • defaultUnit: String; There is no default; The unit of Spatial Reference.
    • isGeographicCS: Boolean; There is no default; Indicates whether the spatial reference is a Geographic Coordinate System or not.
    • isGeographicUnit: Boolean; There is no default; Indicates whether the output unit is Geographic Coordinate Unit or not.
    • isProjectedCS: Boolean; There is no default; Indicates whether the spatial reference is a Projected Coordinate System or not.
    • isProjectUnit: Boolean; There is no default; Indicates whether the output unit is Projected Coordinate Unit or not.
    • sameSRWithMap: Boolean; There is no default; Indicates whether the output wkid is the same as the map.
    • spheroidCS: Number; There is no default; The wkid of reference ellipsoid.
    • unitRate: Number; There is no default; The rate from default unit to output unit.
  • transformForward: boolean. There is no default. Indicates whether to transform forward or not. The forward or reverse direction of transformation is implied in the name of the transformation.
  • outputunit: String. Valid values are Default,INCHES,FOOT,YARDS,MILES,NAUTICAL_MILES,MILLIMETERS,CENTIMETERS,METER,KILOMETERS,DECIMETERS,DECIMAL_DEGREES,DEGREE_MINUTE_SECONDS,MGRS,USNG. The default value is an empty string, which means using the default unit of wkid.
decimalPlaces

Number; The numeric precision of the coordinates.
addSeparator

Boolean; Default is false. Whether to display thousand separators.

Example:

{
  addSeparator: true,
  decimalPlaces: 3,
  "spatialReferences": [{
    "wkid": 3857,
    "label": "WGS_1984_Web_Mercator_Auxiliary_Sphere",
    "transformForward": false,
    "transformationLabel": "",
    "transformationWkid": "",
    "options": {
      "defaultUnit": "Meter",
      "isGeographicCS": false,
      "isGeographicUnit": false,
      "isProjectUnit": true,
      "isProjectedCS": true,
      "sameSRWithMap": true,
      "spheroidCS": 4326,
      "unitRate": 1
    },
    "outputUnit": "Meter"
  },{
    "wkid": 2285,
    "label": "NAD_1983_StatePlane_Washington_North_FIPS_4601_Feet",
    "transformForward": true,
    "transformationLabel": "WGS_1984_(ITRF00)_To_NAD_1983",
    "transformationWkid": 108190,
    "options": {
      "defaultUnit": "Foot_US",
      "isGeographicCS": false,
      "isGeographicUnit": false,
      "isProjectUnit": true,
      "isProjectedCS": true,
      "sameSRWithMap": false,
      "spheroidCS": 4269,
      "unitRate": 0.00018939431818257576
    },
    "outputUnit": "MILES"
  }]
}
