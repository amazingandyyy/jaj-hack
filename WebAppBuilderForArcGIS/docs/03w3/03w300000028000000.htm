Edit

The Edit widget provides editing capabilities using an editable layer in a feature service. It combines the TemplatePicker, AttachmentEditor, AttributeInspector, and GeometryService to provide feature and attribute editing.

The following table shows the descriptions of the configurable attributes.

Attribute

Description

editor

An object of ArcGIS API for JavaScript. Refer to the params of Editor constructor at https://developers.arcgis.com/en/javascript/jsapi/editor-amd.html#editor1.

  • layerInfos—Object. There is no default. The feature layer information.

    featureLayer—Object. There is no default. Reference to the feature layer.

    id—String. There is no default. The id of an editable layer in web map. The layer is ignored if the id does not exist in web map.

  • TableInfos—Object. There is no default. The feature table information.

    featureLayer—Object. There is no default. Reference to the feature table.

    id—String. There is no default. The id of an editable table in web map. The table is ignored if the id does not exist in web map.

  • useFilterEdit—Boolean. Enable templatePicker to filter templates.
  • autoApplyEditWhenGeometryIsMoved—Boolean. Apply the edit automatically when the geometry is moved.
  • snappingTolerance—Number. Set the snapping tolerance in pixels.
  • popupTolerance—Number. Set the tolerance of attribute editing popup in pixels.
  • stickyMoveTolerance—Number. Set the sticky move tolerance in pixels.

Example: 

{
  "editor": {
    "enableUndoRedo": true,
    "toolbarVisible": true,
    "toolbarOptions": {
      "mergeVisible": true,
      "cutVisible": true,
      "reshapeVisible": true
    },
    "layerInfos": [{
      "featureLayer": {
       "id":"Wildfire_4234"
      },
      "disableGeometryUpdate": true,
      "fieldInfos": [{
        "fieldName": "rotation",
        "isEditable": false
      }, {
        "fieldName": "description",
        "isEditable": true
      }, {
        "fieldName": "eventdate",
        "label": "Event Date",
        "isEditable": false
      }, {
        "fieldName": "eventtype",
        "label": "Event Type",
        "isEditable": false
      }]
    }, {
      "featureLayer": {
        "id":"Wildfire_6333"
      },
      "disableGeometryUpdate": true,
      "fieldInfos": [{
        "fieldName": "symbolid",
        "isEditable": false
      }, {
        "fieldName": "description",
        "isEditable": true
      }]
    }],
    "tableInfos": [],
    "useFilterEdit": false,
    "autoApplyEditWhenGeometryIsMoved": false,
    "snappingTolerance": 15,
    "popupTolerance": 5,
    "stickyMoveTolerance": 0
  }
}
