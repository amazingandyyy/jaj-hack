Screening widget

The Screening widget allows you to define an area of interest and analyze specified layers for potential impacts. The area of interest can be defined through a place name search, by drawing on the map, uploading a shapefile, or entering a traverse with coordinate and distance pairs. Analysis results can be inspected in the widget and optionally shared via a printed report, CSV file download, or feature download.

Configuring the Screening widget

The Screening widget can be set to open automatically when the app starts. To do so, click the dot on the widget to turn it dark green.

Steps:
  1. Hover over the widget and click the edit icon .

    The configuration window for the widget appears.

  2. Optionally click the change widget icon, and replace the icon with your own image.

    A file explorer window appears, allowing you to select a local image file to use as the widget icon.

  3. Click Add Layer. Each added layer specifies a feature layer in the map to be analyzed by the widget. The following layer types are supported with the Screening widget:
    • Feature services
    • Map services
  4. On the Analysis tab, optionally add a label for each layer.
  5. Click the small edit icon under the Actions header to select the fields to be presented with the analysis results.
  6. Under Actions, you can change the order of layers or delete a layer.
  7. Choose which tools will be available to define the area of interest under AOI Tools and update the label.
  8. Enable Allow uploading shapefiles to analysis. With this option, users will be able to optionally add an additional layer for analysis from within the widget.
  9. For Show areas/lengths in, specify the default units in which areas and lengths will be shown by choosing Standard Unit (miles and acres) or Metric Unit (kilometers and square kilometers).
  10. Specify input for Max features for analysis. This parameter limits the maximum number of features returned from the configured layers and used for analysis.
  11. Specify input for Search tolerance in feet. The search tolerance will be used to select features when the area of interest is a point or line shape.
  12. Click the Download tab, and under Download Settings, choose one of the following options:
    1. Feature Layers—Choose the layers allowed to be downloaded. This option supports CSV file and, when the layer is a feature service with sync enabled, file geodatabase downloads.
    2. Extract Data Task—All layers are enabled for CSV file download and feature data can be downloaded as file geodatabase or shapefile with the Extract Data Task geoprocessing service
    3. Disable download—No data download is enabled with this option
  13. Under Report Settings, do the following:
    1. Update Print Service URL to use an alternate print service with the Export Web Map task enabled.
    2. Use the Logo option to change the logo by clicking the image.

      A file explorer window appears, allowing you to select a local image file to use as the logo.

    3. Change the default Report Title text.
    4. Change the Color for column titles option. This color will be displayed as the background for column titles in the printed report.
    5. Enter text in the Footnote text box that will be displayed as a footnote on the printed report.
  14. Check the General tab. Update the labels used in the widget for the Area of Interest and Report panels.
  15. Update the default buffer settings for the widget.
  16. Change the symbology used to depict the buffer in the map
  17. Change the symbology used to depict the point, line, or polygon areas of interest in the map.
  18. Check the Search Source Settings tab. To configure the Search Source Settings section, see Configuring the Search widget. It follows the same configuration steps as the Screening widget.

Using the Screening widget

Steps:
  1. Click the Screening widget icon to open it. The widget displays on the right side of the map.
  2. Define an area of interest with one of the provided tools.
    • Placename—Search for a location in the map, and optionally apply a buffer.
    • Draw—Draw a point, line, or polygon on the map or select a feature from the map, and optionally apply a buffer. Use the Choose selectable layer list to define which layers are selectable in the map.

    • Shapefile—Upload a zipped shapefile, and optionally apply a buffer.
    • Coordinates—Identify a starting point for a traverse by searching the map, entering a coordinate pair, or drawing a point on the map. Enter bearing and distance pairs to define the traverse.

  3. Define a buffer.
  4. Click Start Over to remove the defined area of interest and create a new one.
  5. Click Report to analyze the configured layers within the area of interest.

    Analysis results are presented in the panel. Layers with data intersecting the area of interest report a total feature count, and results are presented by grouping unique attribute combinations with the following information:

    • Area of overlap when the area of interest is a polygon
    • Length of overlap when the area of interest is a polygon

    Change the units in which measurements are reported by toggling between Metric and Standard.

Explore the results

Steps:
  1. Expand the result details by clicking a layer. The layer details expand as shown in the following screen shot:

  2. Click the gear icon to choose which fields are presented with the results.

  3. Add an additional layer for analysis by clicking the Upload button.

    A file explorer window appears, allowing you to select a local zipped shapefile.

  4. Download feature information in CSV file, file geodatabase, or shapefile format by clicking the Download button and choosing a format.

  5. Print a report of the analysis results by clicking the Print button and choosing a page size and orientation.

    A new window appears. Change the report title or comments.
